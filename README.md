1. **Flutter BLoC** for State Management.
2. **Auto Router** (Routing package).
3. **Get It** (Service Locater and Dependencies container).
4. **Dio** (Http Client).
5. **BLoC Test**.
6. **Http Mock Adapter** for testing.
